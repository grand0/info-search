## Задание 1

`collect_links.py` собирает список новостей с сайта https://kod.ru и сохраняет ссылки в файл `urls.txt` для последующего скачивания и парсинга.
`main.py` считывает ссылки из файла `urls.txt`, считывает содержимое из каждой ссылки и записывает его под своим номером в подпапку `articles`, а также записывает номер ссылки и саму ссылку в файл `index.txt`
