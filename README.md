## Задание 1

`collect_links.py` собирает список новостей с сайта https://kod.ru и сохраняет ссылки в файл `urls.txt` для последующего скачивания и парсинга.

`read_articles.py` считывает ссылки из файла `urls.txt`, считывает содержимое из каждой ссылки и записывает его под своим номером в подпапку `articles`, а также записывает номер ссылки и саму ссылку в файл `index.txt`


## Задание 2

`tokenize_articles.py` проходит по статьям, находит токены и лемматизирует их. Списки токенов сохраняются в папке `tokens`, списки лемм в папке `lemmas`.

## Задание 3

`index.py` использует леммы из папки `lemmas` и сопоставляет их номерам документов, составляя таким образом индекс темринов, сохраняемый в файл `termin_index.txt`

`bool_search.py` реализует булевый поиск по созданному индексу терминов и возвращает список документов, подходящих под запрос
