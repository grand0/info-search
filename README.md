# Основы информационного поиска

- Парамонов Дмитрий 11-201
- Пономарев Илья 11-201


## Задание 1

`collect_links.py` собирает список новостей с сайта https://kod.ru и сохраняет ссылки в файл `urls.txt` для последующего скачивания и парсинга.

`read_articles.py` считывает ссылки из файла `urls.txt`, считывает содержимое из каждой ссылки и записывает его под своим номером в подпапку `articles`, а также записывает номер ссылки и саму ссылку в файл `index.txt`


## Задание 2

`tokenize_articles.py` проходит по статьям, находит токены и лемматизирует их. Списки токенов сохраняются в папке `tokens`, списки лемм в папке `lemmas`.

## Задание 3

`index.py` использует леммы из папки `lemmas` и сопоставляет их номерам документов, составляя таким образом индекс терминов, сохраняемый в файл `termin_index.txt`

`bool_search.py` реализует булевый поиск по созданному индексу терминов и возвращает список документов, подходящих под запрос

## Задание 4

`tf-idf.py` вычисляет tf-idf для токенов и лемм и сохраняет их в папках `tf-idf-tokens` и `tf-idf-lemmas` соответственно

## Задание 5

`vector_search.py` выполняет векторный поиск используя рассчитанные tf-idf и возвращает список найденных статей.

`web_server.py` - веб-сервер для демонстрации работоспособности векторного поиска. Выдаёт страницу `index.html` с полем поиска и списком найденных статей.
