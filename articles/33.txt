<div><p>Читайте в Telegram</p><p>Apple выпустила релизные версии iOS 26.3, iPadOS 26.3 и macOS 26.3 для всех пользователей.</p><p>Рассказываем, что в них появилось нового.</p><h2 id="%D0%BE%D0%B1%D0%BD%D0%BE%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F-ios-263-%D0%B8-ipados-263">Обновления iOS 26.3 и iPadOS 26.3</h2><ul>
<li>Появился перенос данных с iPhone на Android «по воздуху»: достаточно разместить рядом Android-смартфон с iPhone, чтобы перенести фото, сообщения, заметки, приложения, номер телефона и прочее.</li>
</ul>
<blockquote>Для этого необходимо открыть «Настройки» → «Основные» → «Перенос или сброс iPhone» → «Перенос на Android».</blockquote><img src="https://cdn.kod.ru/rs:fill/w:850/q:85/plain/https%3A%2F%2Fapi.kod.ru%2Fcontent%2Fimages%2F2026%2F02%2Fappl-1.png@webp" class="kg-image" alt="Apple выпустила iOS 26.3 и macOS 26.3: что нового" width="850" height="582"><ul>
<li>В странах ЕС появилась функция упрощённого сопряжения находящихся рядом аксессуаров от сторонних производителей, например, наушников или часов. Благодаря ней подключить сторонний аксессуар можно будет так же легко, как «родные» AirPods.</li>
</ul>
<ul>
<li>В странах ЕС также появилась функция переадресации уведомлений, которая позволяет пересылать уведомления с iPhone на сторонние умные часы и браслеты, а не только на Apple Watch. Обе функции внедрены для соответствия Закону ЕС о цифровых рынках.</li>
</ul>
<ul>
<li>Переработан раздел выбора обоев — категории «Погода» и «Астрономия» разделили, для «Погоды» добавили несколько вариантов погодных обоев с пресетами, настройкой виджетов и стиля часов на экране блокировке.</li>
</ul>
<img src="https://cdn.kod.ru/rs:fill/w:850/q:85/plain/https%3A%2F%2Fapi.kod.ru%2Fcontent%2Fimages%2F2026%2F02%2Fappl-2.png@webp" class="kg-image" alt="Apple выпустила iOS 26.3 и macOS 26.3: что нового" width="850" height="582"><ul>
<li>На устройствах с модемами Apple (iPhone Air, iPhone 16e и iPad Pro на M5) появился отдельный переключатель, позволяющий ограничить операторам связи определение точного местоположения пользователя: вместо адреса смартфон будет сообщать только приблизительный район. Для активации функции требуется поддержка на уровне оператора.</li>
</ul>
<ul>
<li>Apple также внедрила некоторые изменения «под капотом», которые в переспективе позволят операторам включать сквозное шифрование (E2EE) для сообщений RCS на iPhone — стандарта, позволяющего вести кросс-платформенную переписку между устройствами Android и iOS.</li>
</ul>
<h2 id="%D0%BE%D0%B1%D0%BD%D0%BE%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5-macos-263">Обновление macOS 26.3</h2><ul>
<li>В обновлённой версии браузера Safari 26.3 для новой macOS исправили проблемы с Video Viewer mode для iframe-видео: то есть были убраны баги при просмотре роликов, которые встроены в сайты, например, из YouTube.</li>
</ul>
<ul>
<li>Внедрено фоновое обновление настроек безопасности для Safari, WebKit и некоторых системных библиотек между выходами обновлений ОС. Эта функция получила название Background Security Improvements.</li>
</ul>
<ul>
<li>Ряд изменений для AppKit, StoreKit и внедрений новшеств «под капотом» для грядущих MacBook Pro на M5 Pro и M5 Max.</li>
</ul>
<blockquote>Обновление уже доступно всем пользователям. Откройте «Настройки» → «Основные» → «Обновление ПО» и скачайте новую версию.</blockquote></div>
