<div><p>Функция историй появилась в мессенджере в июле 2023 года, однако с тех пор пользователи из ряда стран, которые не имеют подписку Telegram Premium, не могли публиковать их.</p><p>Среди стран, в которых действовало ограничение, была и Россия. С 6 февраля 2026 года Telegram открыл для россиян возможность публикации историй без необходимости подключать для этого подписку Telegram Premium. </p><img src="https://cdn.kod.ru/rs:fill/w:850/q:85/plain/https%3A%2F%2Fapi.kod.ru%2Fcontent%2Fimages%2F2026%2F02%2Ftest-1.png@webp" class="kg-image" alt="Пользователи из России теперь могут публиковать истории без Telegram Premium" width="850" height="582"><ul>
<li>«Код Дурова» убедился в том, что ограничение для аккаунта без подписки было снято.</li>
</ul>
<ul>
<li>По данным «Кода Дурова», в день можно будет публиковать только одну историю: при оформлении Telegram Premium можно увеличить это количество до 100.</li>
</ul>
<img src="https://cdn.kod.ru/rs:fill/w:850/q:85/plain/https%3A%2F%2Fapi.kod.ru%2Fcontent%2Fimages%2F2026%2F02%2Ftest-2.png@webp" class="kg-image" alt="Пользователи из России теперь могут публиковать истории без Telegram Premium" width="850" height="582"><ul>
<li>Ограничение на публикацию историй без Telegram Premium было снято и для аккаунтов с номерами +380 и +98 — Украина и Иран.</li>
</ul>
<ul>
<li>Если вы опубликовали истории за всё время 3 раза, то больше не сможете делать это без оформления Telegram Premium.</li>
</ul>
